概述 / OVERVIEW

CR400AF是中国铁路新一代高速动车组的代表车型之一，隶属于"复兴号"系列。由中车青岛四方机车车辆股份有限公司研制，于2017年6月正式投入运营。作为中国达到世界领先水平的高速列车，复兴号实现了从"跟跑"到"领跑"的历史性跨越。

CR400AF在设计上充分体现了中国自主创新的成果，其整车设计及关键部件均拥有完全自主知识产权。列车采用了全新的低阻力流线型头型，进一步降低了空气阻力；车体结构更加轻量化，同时提升了气密性和隔音性能，为旅客提供更为舒适的乘坐体验。

历史沿革 / HISTORY
2013

中国铁路总公司正式启动新一代高速动车组研制项目，代号"复兴"。

2015

四方股份完成首列CR400AF样车制造，开始进行线路试验。

2016.07

在郑徐高铁上进行时速420km/h交会试验，成功创造世界纪录。

2017.06

CR400AF在京沪高铁正式投入商业运营，标志着中国高铁进入"复兴号"时代。

CR400AF在京沪高铁全线投入商业运营 / 图：中国铁路
设计特征 / DESIGN FEATURES

CR400AF的头型设计灵感来源于"飞龙"意象，流线型曲线从前端向后延伸，形成极具辨识度的视觉特征。相较于CRH380系列，复兴号的头型降低了约12%的空气阻力系数。

配属与运营 / DEPLOYMENT

CR400AF目前主要配属北京局、上海局和广州局集团公司，承担京沪高铁、京广高铁等核心干线的运营任务。列车编组以8辆标准编组为主，也可重联运行形成16辆编组，以满足高峰时段的运输需求。
